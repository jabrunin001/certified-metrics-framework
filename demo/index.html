The certification contract

Three gates between a query and a trusted number

Schema tests prove a column is the right shape. They cannot prove the logic is right. Certification adds the gate that does.

1

Govern

One definition in MetricFlow, one named owner, registered in source control. No more three teams with three versions of "active user."

2

Check freshness

Every source partition must land inside its SLA window before the metric is eligible. Stale inputs never quietly become a stale number on a slide.

3

Reconcile

The pipeline value is compared to a golden value re-derived down an independent path. Disagreement beyond tolerance fails the build and names the metric, the variance, and the likely cause.
